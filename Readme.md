# Clover Pit

Final Unity project for Clover Pit.

## Unity Version

Open with Unity `6000.2.6f2`.

## Project Structure

- `Assets/Scenes`: main Unity scenes. The main scene is `MainScene`.
- `Assets/Scripts`: gameplay systems grouped by feature.
- `Assets/Prefabs`: reusable Unity objects.
- `Assets/ScriptableObjects`: data assets used by gameplay systems.
- `Assets/Materials`, `Assets/Sprites`, `Assets/Audio`, `Assets/Fonts`: project media and presentation assets.
- `Packages`: Unity package manifest and lock file.
- `ProjectSettings`: Unity project settings required to open the project correctly.

## Notes

Generated Unity folders such as `Library`, `Temp`, `obj`, `Logs`, and `UserSettings` are intentionally ignored by Git. Unity will recreate them locally when the project is opened.
